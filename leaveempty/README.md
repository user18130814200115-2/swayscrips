# Leave Empty
After quitting the last window in a workspace,
move to the previous workspace by number.

## NOTE
- Quite untested
- Does not work with named workspaces (will add if requested)

## Dependencies
- Posix shell
- expr
- [lswt](https://git.sr.ht/~leon_plickat/lswt)
- grep
- cut
- sway (obv)
